# 公开课学习总结

这里是个人学习公开课的笔记和总结仓库。

## 项目结构

```
.
├── README.md                        # 项目说明
├── templates/                       # 学习模板
│   ├── course-template.md          # 单门课程模板
│   └── weekly-template.md          # 周学习计划模板
├── courses/                         # 各课程学习笔记
│   └── course-name/                # 课程目录（以课程名命名）
│       ├── 00-course-info.md       # 课程基本信息
│       ├── 01-weeks/               # 按周组织的笔记
│       ├── 02-projects/            # 项目实践
│       └── 03-resources/           # 参考资料
└── notes/                          # 日常学习笔记

```

## 学习流程

1. **课程开始前**
   - 创建课程目录
   - 填写课程信息
   - 制定学习计划

2. **学习过程中**
   - 按周记录笔记
   - 完成编程练习
   - 整理重点知识

3. **课程结束后**
   - 总结核心要点
   - 整理项目作品
   - 写学习反思

## 笔记规范

- 使用 Markdown 格式
- 代码块标注语言
- 重点内容用引用块标记
- 定期整理和回顾

## MCP 服务器集成

本项目已集成 5 个开箱即用的 Model Context Protocol (MCP) 服务器，增强了 AI 辅助学习能力：

### 已配置的 MCP 服务器（无需 API 密钥）

1. **📁 Filesystem** - 本地文件操作和管理
2. **🌐 Fetch** - 网页内容获取和抓取
3. **🧠 Memory** - 知识图谱和记忆管理
4. **⏰ Time** - 时间工具和日期计算
5. **📄 PDF Reader** - PDF 文档读取和分析

详细配置和使用指南请查看: [notes/MCP-SERVERS.md](notes/MCP-SERVERS.md)

### 快速开始

这些 MCP 服务器已配置完成，可直接在 Claude Code 中使用，无需额外设置 API 密钥。
