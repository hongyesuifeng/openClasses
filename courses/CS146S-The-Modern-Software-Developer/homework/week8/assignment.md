## 作业概述
在 3 个不同的技术栈中构建相同功能的 Web 应用。至少有一个版本必须使用 [`bolt.new`](https://bolt.new/)（一个 AI 应用生成平台）创建。至少有一个版本必须对前端或后端使用非 JavaScript 语言（例如 Django、Ruby on Rails）。

你可以重用前几周的应用（"开发者指挥中心"）或创建你选择的新应用，只要它满足[最低功能范围](#最低功能范围)。应用应该是端到端功能性的（前端 + 后端 + 适用时的持久化），并展示一个连贯的功能集。

## 最低功能范围
- 用户可以创建、读取、更新和删除主要资源（例如，笔记、任务、帖子）。
- 适用的持久化存储（数据库或基于文件）。
- 基本验证和错误处理。
- 简单但实用的 UI，展示主要流程。
- 清晰的说明以在本地运行每个版本（如果你部署，还有部署链接）。

## 技术栈要求
构建同一应用的 3 个独立版本，每个使用不同的技术栈。例如：
- MERN (MongoDB、Express、React、Node.js)
- MEVN (MongoDB、Express、Vue.js、Node.js)
- Django + React（或 Vue）
- Flask + Vanilla JS（或 React）
- Next.js + Node（或 NestJS）
- Ruby on Rails（全栈）

提醒，至少有一个版本必须对前端或后端包含非 JavaScript 语言（例如 Python/Django、Ruby/Rails）。


至少有一个版本必须使用 AI 应用生成平台 **[`bolt.new`](https://bolt.new/)** 构建，但你可以随意探索其他应用生成平台（例如 Lovable、Figma Make）来构建其他版本。


## 了解 Bolt
Bolt 是一个 AI 辅助开发平台，可从自然语言提示词生成网站、Web 应用和移动应用。用户可以用纯文本描述他们的想法，Bolt 会在几分钟内生成功能原型——从落地页和电子商务网站到 CRM 和移动工具。了解更多[此处](https://support.bolt.new/building/intro-bolt)。

### 领取你的 Bolt 积分：
1. 找到你通过电子邮件收到的唯一 Bolt 促销代码。
2. 导航到 [bolt.new](bolt.new) 并创建一个帐户。
3. 在个人设置 > 订阅和令牌中，在升级到 Pro 块中，点击蓝色的"升级"按钮。
4. 选择"添加促销代码"并将你的唯一促销代码粘贴到此字段中。
5. 你将免费获得 3 个月的 Bolt Pro。需要信用卡来激活试用。**记住在 3 个月期限结束前取消，以避免在你不打算继续订阅的情况下自动计费。**


## AI 应用生成器使用提示
- 像 Bolt 这样的应用生成器最适合现代全栈技术，当你在不指定特定框架的情况下使用它们时，你将默认获得这些技术。
- 优先使用描述你的应用概念、实体、路由和 UI 流程的清晰提示词。
- 在提示词中清晰描述数据模型和关系。
- 迭代改进数据模型、CRUD 端点、auth（如果使用）和前端组件。
- 保持每个版本独立，以避免依赖冲突。
- 导出或同步生成的代码，并将其作为该技术栈的独立项目文件夹提交。
