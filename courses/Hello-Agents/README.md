# Hello-Agents 智能体系统学习教程

> 从零开始构建智能体 - 系统性学习AI Agent的完整指南

## 课程简介

Hello-Agents 是 Datawhale 社区的系统性智能体学习教程，专注于 AI Native Agent 的构建。本教程将带领你从智能体的核心原理出发，深入其核心架构，理解其经典范式，并最终亲手构建属于自己的多智能体应用。

## 学习目标

- 理解智能体的核心原理、历史与经典范式
- 掌握热门低代码平台和智能体代码框架的使用
- 基于 OpenAI 原生 API 从零构建自己的智能体框架
- 掌握上下文工程、Memory、协议、评估等系统性技术
- 学习 Agentic RL，从 SFT 到 GRPO 的全流程实战训练 LLM
- 实战开发智能旅行助手、赛博小镇等综合项目

## 重点拓展方向

### 🎮 游戏开发应用
智能体在游戏开发中的应用是当前最热门的方向之一：

- **NPC 智能**：创建具有独立人格和行为的非玩家角色
- **游戏测试**：使用智能体自动化测试游戏逻辑和平衡性
- **内容生成**：自动生成游戏剧情、对话、任务等
- **赛博小镇**：模拟社会动态的多智能体系统

具体应用案例：
- RPG 游戏中的智能对话系统
- 策略游戏中的 AI 对手
- 开放世界游戏中的动态事件生成
- 多人游戏中的智能陪练

### 💻 AI编程助手
智能体在编程领域的应用：

- **代码生成**：根据需求自动生成代码
- **代码审查**：智能代码质量检查和优化建议
- **Bug 修复**：自动定位和修复代码问题
- **文档生成**：自动生成技术文档和注释
- **测试生成**：自动生成单元测试和集成测试

## 学习路径建议

1. **基础阶段**（第1-3章）：理解智能体概念和LLM基础
2. **实践阶段**（第4-7章）：动手实现经典范式和框架
3. **进阶阶段**（第8-12章）：掌握高级技术和系统设计
4. **项目阶段**（第13-15章）：完成综合实战项目
5. **创作阶段**（第16章+）：构建自己的多智能体应用
