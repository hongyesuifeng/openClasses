## 📖 如何生成PDF电子书

### 方法1：使用在线转换工具（推荐，最简单）

#### A. Markdown to PDF
1. 访问: https://www.markdowntopdf.com/
2. 上传文件: `Hello-Agents-完整课程.md`
3. 点击"Convert"按钮
4. 下载生成的PDF文件

#### B. CloudConvert
1. 访问: https://cloudconvert.com/md-to-pdf
2. 上传Markdown文件
3. 转换并下载PDF

#### C. Dillinger
1. 访问: https://dillinger.io/
2. 导入Markdown文件
3. 导出为PDF

---

### 方法2：使用VS Code插件

1. 打开VS Code
2. 安装插件: "Markdown PDF"
3. 打开 `Hello-Agents-完整课程.md`
4. 右键选择: "Markdown PDF: Export (pdf)"

---

### 方法3：使用Typora（推荐，效果最好）

1. 下载并安装Typora: https://typora.io/
2. 用Typora打开 `Hello-Agents-完整课程.md`
3. 菜单: 文件 → 导出 → PDF
4. 调整PDF设置（建议使用A4纸张）

---

### 方法4：使用Pandoc（命令行工具）

如果您有Pandoc，运行以下命令：

```bash
cd "/mnt/c/Users/Administrator/Desktop/公开课/openClasses/courses/Hello-Agents"

pandoc Hello-Agents-完整课程.md -o Hello-Agents-完整课程.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="Microsoft YaHei" \
  -V mainfont="Microsoft YaHei" \
  --toc \
  --number-sections \
  --highlight-style=tango
```

---

### 方法5：使用Chrome/Edge浏览器

如果生成了HTML版本：

1. 在浏览器中打开HTML文件
2. 按 `Ctrl + P` (打印)
3. 目标打印机选择: "另存为PDF"
4. 点击保存
