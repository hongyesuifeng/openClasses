# Hello-Agents 学习路径与时间规划

## 整体学习周期

建议总学习时间：**8-12周**（根据个人基础和投入时间调整）

## 分阶段学习计划

### 第一阶段：基础认知（1-2周）

**目标**：建立智能体和LLM的基础认知

#### Week 1: 智能体基础
- **第1章：初识智能体**
  - 智能体的定义和核心特征
  - 智能体的类型分类
  - Agent vs LLM 的区别
  - 智能体的应用场景
  - 游戏中的智能体应用（重点）

- **第2章：智能体发展史**
  - 符号主义智能体
  - 强化学习智能体
  - 大语言模型驱动的智能体
  - 未来发展趋势

**实践作业**：
- 分析3个不同类型的智能体应用
- 设计一个简单的智能体概念（游戏NPC或编程助手）

#### Week 2: LLM基础
- **第3章：大语言模型基础**
  - Transformer 架构
  - 提示工程基础
  - 主流 LLM 对比（GPT、Claude、Llama等）
  - LLM 的局限性
  - LLM API 调用实践

**实践作业**：
- 调用至少2个不同的LLM API
- 实现基础的提示工程模板

## 学习建议

### 每周学习时间分配
- 理论学习：30%
- 代码实践：50%
- 项目实战：20%

### 学习方法
1. **先通读后精读**：先快速了解整体，再深入研究细节
2. **边学边做**：每学完一个概念立即动手实现
3. **记录总结**：建立自己的学习笔记和知识体系
4. **社区交流**：参与讨论，分享经验

### 常见问题
- **时间不够**：可以延长到16周，重点放在实践项目
- **基础薄弱**：先补充 Python 和 LLM 基础知识
- **项目选择**：根据兴趣选择游戏开发或AI编程方向
