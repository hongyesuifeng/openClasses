## 学习路径建议

### 初学者路径
1. **GAMES101** → 学习图形学基础
2. **3D数学基础** → 掌握数学工具
3. **OpenGL/Vulkan Tutorial** → 学习图形API
4. **GAMES104** → 学习引擎架构
5. **游戏引擎架构** → 深入理解设计

### 进阶路径
1. **Real-Time Rendering** → 系统学习渲染
2. **Jason Gregory书籍** → 引擎架构深入
3. **阅读引擎源码** → Piccolo, Unreal, Unity
4. **实现Demo** → 实践项目
