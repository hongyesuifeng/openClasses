## Piccolo引擎目录结构

```
engine/source/
├── runtime/                    # 运行时核心
│   ├── core/                  # 基础设施
│   │   ├── base/             # 基础定义
│   │   ├── math/             # 数学库
│   │   ├── log/              # 日志系统
│   │   └── meta/             # 元数据/反射
│   ├── function/             # 功能系统
│   │   ├── render/           # 渲染系统 ★
│   │   ├── physics/          # 物理系统 ★
│   │   ├── animation/        # 动画系统 ★
│   │   ├── particle/         # 粒子系统 ★
│   │   ├── input/            # 输入系统
│   │   ├── ui/               # UI系统
│   │   ├── framework/        # ECS框架 ★
│   │   └── controller/       # 控制器
│   ├── resource/             # 资源管理 ★
│   └── platform/             # 平台抽象层
└── editor/                    # 编辑器工具 ★
```

## 学习方法

1. **理论与实践结合**: 每看完一讲视频，立即在代码中找到对应模块
2. **代码优先**: 先看代码实现，再回看理论加深理解
3. **调试驱动**: 使用调试器单步执行，观察运行流程
4. **笔记输出**: 每周写学习笔记，整理知识脉络

## 代码阅读顺序

```
engine.cpp (入口)
  ↓
runtime/core/ (基础设施)
  ↓
runtime/function/framework/ (ECS框架)
  ↓
runtime/function/render/ (渲染系统)
  ↓
runtime/function/physics/ (物理系统)
  ↓
其他功能模块
```
