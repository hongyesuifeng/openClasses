# 目录概览

此目录包含斯坦福大学 Mihail Eric 教授的“CS146S：现代软件开发人员”课程材料。该存储库作为知识库，包含与人工智能驱动的软件开发相关的讲义、每周阅读材料、家庭作业和项目模板。

内容主要为中文，并按周模块组织，涵盖从基础 LLM 概念和代理架构到 AI 集成开发环境、安全和 UI 生成的高级实践等主题。

## 关键文件和结构

该存储库的组织结构如下：

-   `courses/CS146S-The-Modern-Software-Developer/`：课程的主目录。
    -   `README.md`：所有课程文档的索引，包括每周摘要、家庭作业和学习路径。
    -   `00-课程概览.md`：课程的全面概述，包括其核心概念、时间表和特邀演讲者。
    -   `week-01/` 到 `week-10/`：每个星期的独立目录，包含摘要和详细阅读材料。
    -   `homework/`：包含所有家庭作业，这些是使用 Claude Code、Warp 和 Semgrep 等工具的实践项目。
-   `templates/`：包含用于构建课程笔记和每周计划的 Markdown 模板。
-   `notes/`：包含一般笔记，包括有关集成模型上下文协议 (MCP) 服务器的指南和信息。
-   `README.md`：整个存储库的主 README，解释了结构和学习过程。

## 用法

此目录旨在供学生和自学者访问和遵循“现代软件开发人员”课程。这些材料通过结构化的学习路径，从理论知识到实际应用，指导用户。

每周的结构有助于系统地学习，家庭作业提供了使用尖端 AI 开发工具的实践经验。`templates` 可用于维护有组织的笔记并跟踪整个课程的进度。
